# 카카오 REST API 키 발급 가이드

## 1. 카카오 개발자 계정 만들기

1. https://developers.kakao.com/ 에 접속
2. 오른쪽 상단 "로그인" 클릭 → 카카오 계정으로 로그인
3. 처음이라면 개발자 약관 동의 필요

## 2. 애플리케이션 생성

1. 로그인 후 "내 애플리케이션" 페이지로 이동
2. "애플리케이션 추가하기" 클릭
3. 앱 이름과 사업자명 입력 (개인 사용이면 본인 이름)
4. "저장" 클릭

## 3. REST API 키 확인

1. 생성된 애플리케이션 선택
2. 왼쪽 메뉴에서 "앱 키" 클릭
3. **"REST API 키"** 를 복사

## 4. pathfinder에 등록

```
pathfinder login kakao -k YOUR_REST_API_KEY
```

또는 대화형으로:
```
pathfinder login kakao
```

## 참고

- REST API 키 하나로 카카오 모빌리티 (길찾기)와 카카오맵 (주소/장소 검색) 모두 사용 가능합니다.
- 일일 호출 제한이 있으나 개인 사용에는 충분합니다 (일 300,000회).
- 키는 외부에 노출되지 않도록 주의하세요.
